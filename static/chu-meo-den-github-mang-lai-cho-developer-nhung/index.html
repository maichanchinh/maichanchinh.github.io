Chú mèo đen (github) mang lại cho developer những gì ?

20 May 2016 on Github, Công Nghệ, Lập Trình, Developer

Trờ lại trong một bài viết mới, tạm gác lại những series để viết về những câu chuyện bên lề xoay quanh cuộc đời coding, hôm nay tôi sẽ chia sẻ với các bạn về những lợi ích của Github mang lại cho developer chúng ta. Cho những bạn nào chưa từng nghe về Github. Nói một cách đơn giản và ngắn gọn GitHub là một dịch vụ lưu trữ, quản lý source code trên mây, sử dụng Source Version Control là Git và hoàn toàn miễn phí. GitHub cung cấp chức năng của một Social Network như là feeds, followers và blabla… để các Developer giao tiếp với nhau.

Ngoài một profile Stackoverflow ấn tượng thì một tài khoản GitHub với nhiều repositories và tham gia đóng góp vào những project đa dạng khác nhau đem đến cho bạn nhiều lợi ích đáng kể, thậm chí có thể thay đổi cả sự nghiệp của bạn. Chắc các bạn đang thắc mắc, làm thế nào mà Github lại có thể ảnh hưởng nhiều đến thế. Đơn giản hãy xem những lợi ích mà Github mang lại cho Developer chúng ta.

Github giúp chứng tỏ bạn là ai.

Chẳng thể phủ nhận những lời hay ý đẹp bạn viết trong CV là cần thiết, nhưng dù gì thì đó cũng chỉ là những lời vô thưởng vô phạt, chẳng thể nói lên được gì nhiều, chỉ có những dòng code nằm trên Github mới là bất diệt. Source code luôn là minh chứng đanh thép để thể hiện bạn là developer thật thụ hay chỉ là thằng chém gió.

Hãy đầu tư cho mình một tài khoản Github thật ấn tượng và đưa đường dẫn vào trong CV, chẳng nhà tuyển dụng nào lại dại dột mà bỏ qua bạn đâu.

Github giúp cải thiện kỹ năng code.

Có hàng ngàn hàng vạn cách để học, học trên Github sẽ là một ý kiến không tồi trong thời đại này. Với hàng vạn open source projects, hàng trăm ngàn contributors, hàng tỉ commit mỗi ngày thì chỉ bằng việc xem, so sánh, học tập từ những thay đổi đó đã đem lại cho bạn hàng tá điều hay để cải thiện kỹ năng code của bản thân mình.

Github là một kho tài nguyên tuyệt vời.

Với chức năng Explore, bạn có thể theo dõi, tìm kiếm những open source projects theo đúng technology pattern mà bạn ưa thích.

Đừng ngần ngại mà không tạo ngay cho mình một tài khoảnGithub, tạo những project của riêng mình và chia sẻ với mọi người, hoặc bạn có thể thoải mái fork một project của một open source nào đó, tạo pull request hoặc issues nếu như tìm được lỗi, cần support.

Và nếu bạn còn bỡ ngỡ chưa biết làm gì sau khi tạo tài khoản Github thì https://help.github.com/ và https://guides.github.com/ sẽ là những thứ vô cùng hữu ích với bạn.
